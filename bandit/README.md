# Bandit

Bandit is the introductory wargame from OverTheWire. It teaches Linux command line basics and is recommended for absolute beginners.

This directory contains notes for each level. Passwords are redacted.

## Connection Information

```
Host: bandit.labs.overthewire.org
Port: 2220
```

Connect via SSH:

```bash
ssh bandit0@bandit.labs.overthewire.org -p 2220
```
